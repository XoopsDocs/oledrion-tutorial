# Blocks

MyShop has several "Blocks" to make it easier for you to run your e-Commerce business:

* Recent Products
* Top Products
* Categories
* Best Sellers
* My Lists
* Lists of current category 
* Module Top Tags
* Recently Sold
* Random lists
* Best rated products
* Module Tag Cloud
* Recommended products
* Ajax search
* Shopping Cart
* Most viewed lists
* Products on promotion
* Random Products
